ENVIRONMENT VARIABLES

Use environment variables to modify a device at run-time. If running from the source tree, just prefix the command with the environment variables:

$ DEMO=true go run ./cmd/

If running from Docker, use --env or -e: 

$ docker run -e DEMO=true merliot/hub

PORT

Port on which the device web server runs. Defaults to 8000.

DEVICES

JSON string or device list for hardware setup.

DEVICES_FILE

File path for device configuration, defaults to devices.json.

USER

Username for authentication or service access.

PASSWD

Password for authentication or service access.

WIFI_SSIDS

Comma-separated list of SSIDs for WiFi setup.

WIFI_PASSPHRASES

Comma-separated list of WiFi passphrases.

DIAL_URLS

Comma-separated URLs for dialing services.

DEMO

Runs the demo mode, or defaults to site mode if SITE is enabled.

SITE

Enables the site mode if set to "true".

DEBUG_KEEP_BUILDS

If set to "true", keeps build artifacts for debugging.
